- stylelint support

  - custom settings
- pre-commit and lint-staged supported

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
